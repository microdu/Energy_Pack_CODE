XDAIS Interface Reference

Modules

The XDAIS interfaces are comprised of the following:

  • XDAIS Types and Constants
  • IALG - XDAIS Algorithm Interface
  • XDAIS Trace Support (Experimental)
  • IRES

The XDM interfaces included in this release are comprised of the following:

  • General definitions
    • XDM - Shared XDM Definitions
  • Video
    • Encoding
      • IVIDENC2 - XDM Video Encoder Interface (preferred)
      • IVIDENC1 - XDM Video Encoder Interface (supported)
    • Decoding
      • IVIDDEC3 - XDM Video Decoder Interface (preferred)
      • IVIDDEC2 - XDM Video Decoder Interface (supported)
  • Imaging
    • IIMGDEC1 - XDM Image Decoder Interface
    • IIMGENC1 - XDM Image Encoder Interface
  • Speech
    • ISPHDEC1 - XDM Speech Decoder Interface
    • ISPHENC1 - XDM Speech Encoder Interface
    • Speech interfaces have also been seperated into those which are common across all variants of speech codecs, and those which apply to a particular class:
      • ISPEECH - XDM Speech Interface
      • ISPEECH_AMR - XDM Speech Interface (AMR)
      • ISPEECH1_EVRC - XDM Speech Interface (EVRC)
      • ISPEECH1_G722 - XDM Speech Interface (G.722)
      • ISPEECH1_G7221 - XDM Speech Interface (G.7221)
      • ISPEECH1_G723 - XDM Speech Interface (G.723)
      • ISPEECH1_G726 - XDM Speech Interface (G.726)
      • ISPEECH1_ILBC - XDM Speech Interface (iLBC)
      • ISPEECH1_PCM - XDM Speech Interface (PCM)
      • ISPEECH1_SMV - XDM Speech Interface (SMV)
      • ISPEECH1_WBAMR - XDM Speech Interface (WBAMR)
  • Audio
    • IAUDDEC1 - XDM Audio Decoder Interface
    • IAUDENC1 - Audio Encoder Interface
  • Other
    • IUNIVERSAL - XDM Beta Universal Algorithm Interface
    • IVIDANALYTICS - XDM Beta Video Analytics Interface
    • IVIDTRANSCODE - XDM Video Transcoder Interface

These older XDAIS and XDM interfaces continue to be supported, but are deprecated. We recommend new development implement the interfaces described above. These older interfaces are documented here:

  • XDAIS
    • IDMA
    • IDMA2
    • IDMA3 - Interface for requesting EDMA3 resources
  • XDM Video
    • IVIDDEC1 - XDM Video Decoder Interface
    • IVIDDEC - XDM Video Decoder Interface
    • IVIDENC - XDM Video Encoder Interface
  • XDM Imaging
    • IIMGDEC - XDM Image Decoder Interface
    • IIMGENC - XDM Image Encoder Interface
  • XDM Speech
    • ISPEECH - XDM Speech Interface
    • ISPHDEC - XDM Speech Decoder Interface
    • ISPHENC - XDM Speech Encoder Interface
  • XDM Audio
    • IAUDDEC - XDM Audio Decoder Interface
    • IAUDENC - XDM Audio Encoder Interface
