Detailed Description

This is the XDM speech interface shared between the various SMV codecs.

enum ISPEECH1_SMV_NoisePreProc

Enabling/disabling noise suppression, and selection of noise suppression if the codec supports multiple noise supression modules.

Enumerator:
ISPEECH1_SMV_NOISEPREPROC_OFF 

Noise suppression is disabled.

ISPEECH1_SMV_NOISEPREPROC_NSA 

Noise suppression is enabled, using option A.

ISPEECH1_SMV_NOISEPREPROC_NSB 

Noise suppression is enabled, using option B.

ISPEECH1_SMV_NOISEPREPROC_DEFAULT 

Default setting.
